model-film

This contains all media model/entity classes used by both the app and the provider system.

In this package, all media content instances extends the abstract com.flixclusive.model.film.Film.kt class. This includes both FilmDetails and FilmSearchItem classes.

FilmDetails class is a parent class of both Movie and TvShow while FilmSearchItem is a single final class only of every search media item results.

Use as a dependency

Step 1. Add the JitPack repository to your build file
